# 📰 News Aggregator - NEWSHUB

## 📌 Overview

The News Aggregator is a lightweight web application that fetches and displays the latest news articles in a clean and user-friendly interface. It allows users to view real-time news from multiple sources in one place using a public news API.

This project demonstrates skills in API integration, asynchronous JavaScript, and responsive front-end development.

---

## ✨ Features

- 🔄 Fetches real-time news using News API  
- 🧾 Clean card-based news layout  
- 📱 Responsive design for different screen sizes  
- ⚡ Fast and lightweight frontend  
- 🌐 Dynamic content rendering using JavaScript  
- 🔍 Easy integration with external APIs

## 🚀 Usage

### ⚠️ Important - To run this project, you must use **your own News API key**.


### Step 1: Get a Free API Key

1. Visit: https://newsapi.org  
2. Sign up for a free account  
3. Generate your API key  

### Step 2: Configure the API Key

Open `script.js` and replace the placeholder:

```js
const API_KEY = "YOUR_API_KEY_HERE";
```
👉 Paste your own API key between the quotes.

⚠️ Do not upload your real API key to public repositories.

### Step 3: Run the Project

1. Clone the repository
```
git clone https://github.com/your-username/news-aggregator.git

```
2. Navigate to the folder
```
cd news-aggregator

```
3. Open index.html in your browser to view real time news
